<product_requirements>
The user requested a full-stack web application for interview preparation. The platform features:
1.  **Free Topics Page**: Displays a list of interview topics. Users can select a topic to view associated questions in an accordion format, with a difficulty filter.
2.  **Goldmine Page (Premium)**: Lists companies with logos and number of questions. Clicking a company reveals interview questions categorized by Project, HR, Technical, and Coding. This page is premium-only, with Razorpay integration for payment. Non-premium users see a preview (companies, structure, but not questions) and are prompted to purchase. Bookmarking questions is available for premium users, including free topic questions.
3.  **Interview Experiences Page**: Admins post experiences for different companies, detailing role, number of rounds, and overall experience. Each experience has a dedicated detail page displaying rounds in an accordion and a status tag (e.g., selected, not selected). The main page includes a search bar for companies.
4.  **Admin Panel**: A production-ready panel to manage all aspects: topics, questions (separated into topic-based and company-based), companies (with Cloudinary image upload for logos), interview experiences (round-wise input, status tags), and user management (e.g., premium users, user list).
5.  **Authentication**: Google Social Login (initially Emergent, then custom Passport.js). Admin access is whitelist-based (specific email).
6.  **Performance/Scalability**: Redis for caching and general optimizations to handle load.
7.  **UI/UX**: Simple, minimalist design using squares and rectangles, adhering to modern design guidelines, 100% responsive across all devices.
</product_requirements>

<key_technical_concepts>
- **Backend**: FastAPI, MongoDB (via MongoEngine/PyMongo), Redis (for caching), Cloudinary (image uploads), Authlib (for Google OAuth).
- **Frontend**: React.js, React Router DOM, Shadcn UI components (Select, Accordion, Dialog, Card, Button, Input, Textarea, Toaster), Tailwind CSS, Axios.
- **Authentication**: Google OAuth 2.0 (custom Passport.js implementation), Session Management.
- **Payment**: Razorpay integration.
- **Deployment**: Kubernetes container environment, Supervisorctl for service management.
</key_technical_concepts>
